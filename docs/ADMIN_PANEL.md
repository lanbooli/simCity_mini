# 管理面板使用说明

管理面板是开发调试工具，用于快速操控游戏状态。避免每次测试都要通过正常游戏流程慢慢积累。

## 开启方式

- **快捷键**: `Ctrl+Shift+M` (在游戏页面中)
- **URL 参数**: `http://localhost:8000/?admin=true`

## 修改能力

### NPC 属性

| 操作 | 说明 |
|------|------|
| 基础属性 | 修改 stamina/speed/strength (1-10) |
| 心情 | 切换 neutral/happy/excited/sad/angry/fear |
| 位置 | 移动到任意场景 |
| 活动 | 修改 current_activity 文本 |
| 性格 | 添加/移除 personality 标签 |
| 头像 | 上传/更换头像图片 |
| 开关 | 激活/停用某个 NPC 进程 |

### 关系数值

| 操作 | 说明 |
|------|------|
| favorability | 设置好感度 (-100~100) |
| familiarity | 设置熟悉度 (0~100) |
| intimacy_comfort | 设置亲密舒适度 (0~100) |
| relationship_type | 手动设置关系类型 |
| jealousy_level | 设置吃醋值 |
| love_eligible | 手动开关心动标记 |
| 一键设置 | 预设组合: 陌生人/朋友/恋人/配偶/仇敌 |

### 游戏状态

| 操作 | 说明 |
|------|------|
| 时间跳转 | 跳到指定 Day/Hour/Minute |
| 时间加速 | 设置游戏时间倍速 (1x/5x/10x/60x) |
| 天气切换 | 晴天/雨天/雪天/阴天 |
| 玩家位置 | 移动玩家到任意场景 |

### 快捷测试触发器

| 操作 | 说明 |
|------|------|
| 强制决策 | 强制指定 NPC 立即执行一次 decide() |
| 触发社交 | 强制同场景两 NPC 开始一次社交交换 |
| 触发告白 | 强制某 NPC 向玩家告白 |
| 触发求婚 | 强制某 NPC 向玩家求婚 |
| 触发吃醋 | 让指定 NPC 目睹玩家与他人亲密 |
| 触发越界 | 模拟玩家对 NPC 越界行为 |
| 重置冷却 | 清除某 NPC 的所有动作冷却 |

### 玩家属性

| 操作 | 说明 |
|------|------|
| 玩家属性 | 修改 stamina/speed/strength |
| 玩家物品 | 添加/移除道具 |

## 安全措施

- 仅在 `DEBUG=true` 或 `ADMIN_ENABLED=true` 时启用
- 生产环境后端 API 返回 404
- 所有修改操作记录日志
- 前端默认隐藏，需主动激活
