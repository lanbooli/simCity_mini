# 玩家命令参考

玩家通过在对话输入框中输入命令来执行动作。命令以 `/` 开头。

## 友好动作

| 命令 | 好感变化 | 关系要求 | 说明 |
|------|---------|---------|------|
| `/微笑` | +1 | 无 | 对 NPC 露出友善的微笑 |
| `/挥手` | +1 | 无 | 向 NPC 挥手致意 |
| `/鼓掌` | +1 | 无 | 为 NPC 鼓掌喝彩 |
| `/赞美` | +2 | 无 | 真诚地赞美 NPC |
| `/摸摸头` | +2 | acquaintance | 轻轻摸摸 NPC 的头 |
| `/送礼` | +3 | acquaintance | 送给 NPC 一份礼物 |
| `/搭肩` | +2 | acquaintance | 拍拍 NPC 的肩膀 |

## 亲密动作

| 命令 | 好感变化 | 关系要求 | 说明 |
|------|---------|---------|------|
| `/拥抱` | +3 | friend | 热情地拥抱 NPC |
| `/牵手` | +3 | friend | 轻轻牵起 NPC 的手 |
| `/摸头` | +3 | friend | 温柔地摸摸 NPC 的头 |
| `/靠肩` | +4 | best_friend | 靠在 NPC 肩上 |
| `/亲脸颊` | +4 | best_friend | 轻轻亲吻 NPC 的脸颊 |
| `/亲吻` | +5 | boyfriend/girlfriend | 亲吻 NPC |
| `/深吻` | +3 | boyfriend/girlfriend | 更深的情感表达 |
| `/依偎` | +3 | boyfriend/girlfriend | 安静地依偎 |
| `/说情话` | +1 | boyfriend/girlfriend | 甜蜜的话语 |
| `/牵手散步` | +1 | boyfriend/girlfriend | 公园场景 bonus |

## 负面动作

| 命令 | 好感变化 | 说明 |
|------|---------|------|
| `/殴打` | -5 (物理判定) | 暴力攻击 NPC |
| `/推搡` | -4 (物理判定) | 用力推 NPC |
| `/绊倒` | -3 (物理判定) | 故意绊倒 NPC |

## 恋爱命令

| 命令 | 说明 |
|------|------|
| `/告白` | 向 NPC 表白。需满足心动阈值 (好感 >= 80, 舒适 >= 70, 熟悉 >= 35) |
| `/求婚` | 向恋人求婚。需满足结婚条件 (好感 >= 90, 舒适 >= 90, 熟悉 >= 60, 成为恋人 >= 1440 tick) |
| `/分手` | 结束恋人关系。NPC 心情变 sad/angry，进入 480 tick 冷静期 |
| `/道歉` | 在冷静期内尝试挽回关系。真诚的对话 +20% 复合率 |

## 动作系统规则

### 递减收益

同一动作对同一 NPC 在 480 tick (8 游戏小时) 内重复使用，效果递减：
- 第 1 次: 100%
- 第 2 次: 70%
- 第 3 次: 40%
- 第 4+ 次: 15%

### 好感度曲线

好感度越接近极端值 (-100 / +100)，变化越慢：
- 0-30 区间: 100% 效果
- 30-60 区间: 80%
- 60-80 区间: 55%
- 80-100 区间: 30%

### 上下文修正

| 条件 | 效果 |
|------|------|
| 深夜 (21:00-06:00) + 亲密动作 | ×1.3 |
| 深夜 + 普通动作 | ×0.7 |
| 动作与场景适配 (公园+散步) | ×1.2 |
| 严肃场景 + 亲密动作 | ×0.7 |
| NPC 心情 happy/excited + 正面动作 | ×1.25 |
| NPC 心情 sad + 安慰类动作 | ×1.3 |
| NPC 心情 angry + 任何正面动作 | ×0.6 |
| 外向 NPC + 社交动作 | ×1.2 |
| 害羞 NPC + 亲密动作 | ×0.7 |
| NPC 工作中 | ×0.8 |

### 物理动作判定 (4 级)

物理动作 (殴打/推搡/绊倒) 基于属性差判定：

| 属性差 (玩家-NPC) | 结果 | 效果倍率 |
|-------------------|------|---------|
| >= 3 | 大成功 (critical_success) | ×1.5 |
| >= 1 | 成功 (success) | ×1.0 |
| == 0 | 险些失败 (narrow_fail) | ×0.7 |
| < 0 | 失败 (failure) | ×0.5 |

## 越界行为

当对 NPC 做出不符合当前关系等级的动作时，触发越界判定：

```
动作要求等级 - 当前关系等级 >= 3 → 越界
```

越界后果：
- NPC 根据性格做出反应 (逃跑/斥责/反击/呼救)
- 目击者传播 gossip → 全镇声望下降
- 严重越界 → NPC 永久回避

## 动作冷却

- 两次动作之间至少间隔 20 游戏分钟
- 超频时 NPC 会产生厌烦回应，好感度不变化
- 每个动作有独立冷却时间 (30-360 tick 不等)

## 对话提示

- NPC 对话由 LLM 实时生成，不是预设文本
- NPC 会引用过去的记忆 (RAG 检索)
- 恋爱关系后 NPC 对话风格会变得更亲密
- 玩家忽略 NPC 的主动互动 3 次后，该 NPC 不再主动互动
