# 约会功能完整设计方案

## 一、整体流程

```
玩家邀请 → NPC决策 → 前往地点
    ↓
┌─ 上半场 · 约会地点（固定90分钟）─────────────────┐
│ 对话框 "💕 正在约会中 — 咖啡店 · 喝咖啡"          │
│ 预设项目决定基础恢复速度                          │
│ 玩家可 /动作 自由互动（LLM 含约会上下文）          │
│ 逐步恢复生理/社交值（每游戏分钟 tick）             │
│ NPC 暂停所有自主行动                             │
│ 玩家离开 → "正在约会中，确定离开？" → 确认则终止    │
└─────────────────────────────────────────────┘
    ↓ 时间到
NPC 邀请回家（概率：好友30%/恋人60%/配偶80%）
    ↓ 接受（5分钟倒计时）            ↓ 超时/拒绝
┌─ 下半场 · NPC家中（无固定时长）───┐   约会结束
│ "💕 正在约会中 — 阳光公寓A · 客厅" │   NPC恢复自主
│ 无预设项目，自由 /动作              │
│ 项目含亲密加成（intimacy）         │
│ 继续逐步恢复                      │
│ NPC 暂停自主                      │
│ 玩家离开 → 确认 → 结束             │
└──────────────────────────────────┘
    ↓ 玩家离开
约会结束 → NPC恢复自主 → 回到原场景
```

## 二、约会地点 & 预设项目（上半场）

| 项目 | 地点 | 饥饿/分钟 | 口渴/分钟 | 精力/分钟 | 社交/分钟 |
|------|------|----------|----------|----------|----------|
| 🍽️ 吃饭 | 餐厅 | +1.0 | +0.3 | — | +0.5 |
| ☕ 喝咖啡 | 咖啡店 | — | +0.7 | +0.3 | +0.3 |
| 🚶 散步 | 公园 | — | — | — | +0.6 |
| 🎬 看电影 | 影院 | +0.2 | +0.2 | +0.5 | +0.4 |
| 🍺 喝酒 | 酒吧 | +0.2 | +0.5 | — | +0.5 |

- 持续 90 游戏分钟
- 好感度 +5~15（关系越好加成越多）

## 四、NPC 决策

| 关系 | 接受约会概率 | 接受回家概率 |
|------|------------|------------|
| 陌生人 | 10% | — |
| 认识 | 40% | — |
| 朋友 | 70% | 30% |
| 好友 | 85% | 40% |
| 恋人 | 95% | 60% |
| 配偶 | 99% | 80% |

- 被拒绝后冷却 10 分钟
- 拒绝回复由 LLM 生成

---

## 五、约束规则

| 对象 | 约会中 | 结束后 |
|------|--------|--------|
| NPC 自主行动 | 暂停 | 恢复 |
| NPC 生理危机 | 约会期间不触发 | 恢复检查 |
| 玩家离开 | 弹窗确认，确认→结束 | 正常 |
| 恢复停止 | 玩家离开 / 约会结束 | — |

## 六、NPC 新增状态字段

```python
self._in_date = False              # 约会中（含上下半场）
self._date_player_id = ""          # 约会的玩家
self._date_phase = ""              # "location" | "home" 
self._date_data = {                # 约会运行时数据
    "scene_id": "",                # 约会所在场景
    "room_name": "",               # 房间（下半场）
    "activity": "",                # 当前项目名
    "elapsed": 0,                  # 已进行分钟数
    "total": 90,                   # 上半场总时长
    "tick_rates": {},              # 恢复速度
}
self._home_invite_sent = False     # 已发回家邀请
self._home_invite_expires = 0      # 邀请过期 tick
```

## 七、LLM 上下文注入

```
【当前状态：正在约会中 — 上半场 · 咖啡店 / NPC家中 · 客厅】
项目：喝咖啡 / 自由互动
关系：好友，好感度：65
约会已进行：30 / 45分钟（上半场30 + 下半场15）
```

`/动作` 和对话都注入此上下文。
